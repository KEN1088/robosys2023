# plusコマンド
![test](https://github.com/ken1088/robosys2023/actions/workflows/test.yml/badge.svg)

*このコマンドはファイルから読み込んだ数字の足し算を計算することができます

# デモ
NUMに数字の列を入力する
NUM
1
2
3
4
5
6
7
8
9

./plus < NUM

55

# ソフトウェア
* python
* テスト済み: 3.7~3.10

# テスト環境
* ubuntu最新版
