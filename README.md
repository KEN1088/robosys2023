# robosys2023

コマンドの説明

大学の授業で作成したものです

使い方

このコマンドでは、複数の数の足し算や引き算を計算することができます
複数の数を内包するファイルを　./plus_stdin ＜ファイル　とすることで計算ができます

##ソフトウェア
* python
  *テスト済み: 3.7~3.10

##テスト環境
* ubuntu 20.04 on windows
